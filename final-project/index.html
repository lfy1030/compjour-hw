Deconding Minna no Data

This project uses data from a crowd-sourced food radioactivity measurement database called Minna no Data from Japan. In order to make sense of the large volumes of data, the data is scraped and taxonomy as well as labels added. Visualizations of the data are then developed...for fun. Check out the menu to explore the data.

Disclaimer: I do not own the data and it is obtains from the Minna no Data. All uses of the data on this site follows the guidelines provided on their website.

This project was developed as part of the Stanford University course COMM277A: Computational Journalism
